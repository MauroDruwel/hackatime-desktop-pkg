# hackatime-desktop-git

An Arch Linux package for [Hackatime Desktop](https://github.com/hackclub/hackatime-desktop), a desktop client for Hackatime.

## Installation

Install from the [Arch User Repository (AUR)](https://aur.archlinux.org/packages/hackatime-desktop-git):

```bash
yay -S hackatime-desktop-git
# or with another AUR helper
```

## Features

This package includes:
- The latest git version of Hackatime Desktop
- Patches to remove the white menu bar and enable window transparency
- Desktop integration with app indicators

## Building from Source

If you prefer to build manually:

```bash
git clone https://github.com/MauroDruwel/hackatime-desktop-pkg.git
cd hackatime-desktop-pkg
makepkg -si
```
